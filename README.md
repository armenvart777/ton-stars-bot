# TON Stars Bot

Telegram-бот для отслеживания покупок Telegram Stars через TON-кошелёк.

Бот подключается к TON API, парсит транзакции с Fragment и считает прибыльность по каждой сделке. Можно задать свою цену продажи и сразу видеть, сколько заработано.

## Что умеет

- Показывает полную статистику по купленным звёздам: количество, затраты в TON/RUB/USDT, прибыль
- Фильтрация по периодам: 7 дней, 30 дней, всё время
- Распределение сделок по размеру, по часам и дням недели
- Анализ курса покупки: минимальный, максимальный, средний
- Сравнение прибыли при разных ценах продажи
- Прогноз дохода на неделю и месяц
- Графики: прибыль по дням, история курса, объёмы, накопительная прибыль
- Выгрузка PDF-отчёта со всеми графиками
- Управление несколькими кошельками
- Список последних транзакций с пагинацией

## Стек

- Python 3.11+
- aiogram 3
- aiohttp
- matplotlib
- TON API (tonapi.io)
- CoinGecko API (курс TON)

## Запуск

```bash
pip install -r requirements.txt
cp .env.example .env
# заполнить .env своими данными
python bot.py
```

## Переменные окружения

| Переменная | Описание |
|---|---|
| `BOT_TOKEN` | Токен бота от @BotFather |
| `TON_WALLET` | Адрес TON-кошелька по умолчанию |
| `TONAPI_BASE` | Базовый URL TON API |
| `ADMIN_IDS` | Telegram ID администраторов через запятую |
| `SELL_PRICE_1` | Первая цена продажи звезды (руб.) |
| `SELL_PRICE_2` | Вторая цена продажи звезды (руб.) |
